Verto Animo

Verto Animo is a 2-player coop game that my team and I worked on for Carnegie Mellon University's game organization, the Game Creation Society.

The game was built on the ideas of teamwork and identity. The player's controls were deliberately scrambled so that the players would be encouraged to communicate with each other in order to progress and succeed.

The game earned 2nd place for best game design and gameplay.

Tools: Unity 3D, Audacity, Photoshop, FireAlpaca

Realms and Bridges

An inspiration of the game, Monument Valley, this game is basically taking the first-person view of what it is like to walk surfaces and view the world at a different angle, like inspired game's player, Ida.

The objective of this project was to create and explore a world that allows a free range of movement and perspective to interpret the environment in ways not possible from normal right-side up standpoints.

The player will have the ability to change the camera's direction and angle to explore the worlds and progress towards the goal, which will be randomly reset and generate a new environment to traverse. The idea is to keep the player moving as each round will generate a different world and provide more color to bring more life into the game.

Language: JavaScript
Tools: three.js

Chip's Challenge Level Editor

Chip's Challenge is an old PC game designed by Chuck Sommerville. Using Tileworld along with a few other tools, I created custom level editor using Google Sheets as a base. The user simply copy and pastes the desired icons in the desired space, and then saves it into a JSON file for conversion into a data file for Tileworld.

Levels developed by the class are played and streamed by Mike Traficante, a Schell Games Senior Engineer. A video link is available for Edward Shin's custom level, which won first place.

Tools: Tileworld, Google Sheets, Sublime, Python, JavaScript, JSON

Edge 2D

Edge 2D is a two-dimensional version of an iPhone game called Edge, by Mobigames. Although the 2D world is not the same as a 3D one, Edge 2D holds most of the experiences the player gets from playing it. Rather than a matter of increasing diffculty, however, Edge 2D progresses by increasing playfulness as the player moves forward.
