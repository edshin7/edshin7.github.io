Chip's Challenge Level Editor

Chip's Challenge is an old PC game designed by Chuck Sommerville. Using Tileworld along with a few other tools, I created custom level editor using Google Sheets as a base. The user simply copy and pastes the desired icons in the desired space, and then saves it into a JSON file for conversion into a data file for Tileworld.

Levels developed by the class are played and streamed by Mike Traficante, a Schell Games Senior Engineer. A video link is available for Edward Shin's custom level, which won first place.

Tools: Tileworld, Google Sheets, Sublime, Python, JavaScript, JSON

Edge 2D

Edge 2D is a two-dimensional version of an iPhone game called Edge, by Mobigames. Although the 2D world is not the same as a 3D one, Edge 2D holds most of the experiences the player gets from playing it. Rather than a matter of increasing diffculty, however, Edge 2D progresses by increasing playfulness as the player moves forward.
